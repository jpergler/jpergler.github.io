What I Do

Web Application Development

My specialization is web application development on ASP.NET platform. I've been developing in many areas of web application from frontend, API, backend to database. I am able to work both with team as well as solo.

Great UX

I believe that great and useful application comes from well-thought-out UX. It is not about the colors, rounded border’s, etc. That’s the UI part. UX is about processes and great work with attention catch. Which includes also the UI part but it should serve mainly the goals the user needs to achieve in the app.

Development Support

I have experience with efficient setup of projects in Jira and Confluence. I believe these tools are great support for development teams and information architecture if configured to the needs of the team. I can help you with analysis of the team needs and configuration of these tools.
